# Portfolio Website - Tiyada Maiwong

This is a professional portfolio website showcasing expertise in Executive Product Leadership, with focus on Digital Advertising, Healthcare, and Pet industries.

## Structure

```
portfolio_website/
├── index.html              # Main landing page
├── css/
│   └── styles.css          # Custom CSS styles
├── js/
│   └── main.js             # JavaScript functionality
├── images/                 # Image assets
├── about/
│   └── index.html          # About page
├── projects/
│   └── index.html          # Projects showcase
├── skills/
│   └── index.html          # Skills, industry focus, and leadership expertise
├── contact/
│   └── index.html          # Contact information and form
└── README.md               # This file
```

## Features

- Responsive design that works on all devices
- Dark/light mode support
- Professional layout with sections for:
  - Executive profile and value proposition
  - Career journey timeline
  - Skills and competencies
  - Industry expertise (Healthcare & Pet)
  - Leadership philosophy
  - Project showcase
- Modern UI with Tailwind CSS
- Interactive elements with JavaScript
- SEO-friendly structure

## Content Integration

This portfolio integrates content from multiple sources:
- Tiyada Maiwong's executive product leadership experience
- Industry focus on Healthcare and Pet sectors
- Leadership and collaboration philosophy
- Skills and expertise in commercial product management
- Digital advertising and ad tech proficiency

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Tailwind CSS
- Google Fonts
- Material Symbols

## Customization

To personalize this portfolio:
1. Update the name and contact information in all HTML files
2. Replace placeholder images with actual photos
3. Modify the color scheme in the Tailwind config
4. Update project details and case studies
5. Customize the skills and experience sections

## Deployment

This is a static website that can be deployed to any web hosting service or CDN. Simply upload the entire portfolio_website folder to your hosting provider.
